### SSC Perma Form Storage

An addon for [Shape Shifter Curse](https://modrinth.com/mod/shape-shifter-curse) (mod for Minecraft 1.20.1) that allows reverting permanent transformation and storing their forms for later use.

For details on how it works, refer to the Modrinth page: https://modrinth.com/mod/ssc-perma-form-storage
